基于github和hexo搭建个人博客

发表于 2017-04-08 | 分类于 github

刚开始学习前段的时候就想着要搭建一个自己的博客，但是一直拖着没有实现。但是学习的过程中愈发觉得总结的重要性，很多东西，学了一遍并没有用，要总结反思。加之最近我哥的提醒，便开始学习个人博客的搭建。一开始我哥提的是jrkyll，但是我之前有看到推荐hexo的比较多，便想着先用hexo搭建。

工具选择

Github

首先，github是必须的，既方便快捷，而且还免费，而且还能巩固git的学习。

hexo

界面简洁，静态网站，加载迅速等优点，缺点是如果不同电脑终端的话，需要都配置一遍，比较麻烦。

开始搭建

  1. 安装hexo

  2. 建blog环境

    打开资源管理器，定位到放项目的目标位置。git bash here

    在目标位置，新建一个zzzzc-blog的目标文件夹。

  3. 预览博客

    这时候打开浏览器，输入localhost：4000；就能看到搭好的博客界面。

  4. 配置信息

    找到zzzzc-blog下的_config-yml文件，打开编辑。
    按照提示更改，作者，标题等基础信息。

  5. 再次预览，是否成功

  6. 安装主题

    进入zzzzc-blog下的themes文件夹

    网上选择喜欢的主题，这里选择GitHub上star比较多的一个主题。

    对hexo-theme-next下的_config-yml进行修改。
    返回zc，修改_config-yml中的theme为hexo-theme-next。
    预览。

  7. 连接github

    对zc文件下的_config-yml进行修改，最后添加最后，
    deploy:
    type： git；
    repo： github.com/gityuzi/zc.git
    branch： master
