第一次听说ajax是刚开始接触前端的时候，同学给我推荐书籍，js高级程序设计，js dom编程艺术，他说她已经看完这些，现在在看ajax。那时候我也不懂。再看dom编程艺术的时候，有说到ajax。但是当时刻意翻过不看了，也看不懂。现在重新学习ajax，终于对此有一定了解。

1. 什么是ajax

1.1 定义

全名：Asynchronous Javascript and XML。翻译过来就是异步的js和xml。即：通过在html页面中的js方法，在不刷新整个页面的情况下，实现和服务端进行交流后返回对应的请求数据，返回文本，xml，json等格式的信息。

1.2 功能

  • 向服务器发出请求，而不加载整个页面
  • 接受和处理服务器中的数据

1.3 XMLHttpRequest对象

ajax就是一个浏览器的一个内建对象即XMLHttpRequest对象。（这里不考虑IE8以下的）

1.3.1 对象属性
属性 描述
readystate 当前http请求状态，只读属性
responseText 服务器返回的请求的信息，或者说从服务器接收到的响应体，文本格式
responseXML 返回的响应信息，解析为XML并作为document对象返回
status 服务器返回的状态码。数字
还有一些不是常见的就不一一列出了 ......
1.3.2 对象事件

|onreadystatechange|readyState属性改变时调用的函数|

1.3.3 对象方法
方法 描述
open 创建一个新的http请求，初始化请求参数，指定请求的方法，URL和验证信息，但并不发送请求
send 发送http请求，使用传递给open的参数，发送给请求体
getAllResponeHeaders、getResponeHeader等 ......

2. ajax实现

2.1 简单ajax封装

function ajaxMethon(url,data,method,success) {
    var httpRequest = new XMLHttpRequest();
    //不考虑IE8以下版本的话就这样直接创建就好了。
    if(method === 'get') {
        //get请求
        if(data) {
            //如果有值
            url += '?';
            url += data;
        } else {

        }
        httpRequest.open(method,url);
        httpRequest.send();
    } else {
        //POST请求
        httpRequest.open(method,url);
        httpRequest.setRequestHeader('Content-Type','application/x-www-form-urlencoded');
        if(data) {
            httpRequest.send(data);
        } else {
            httpRequest.send();
        }
    }
    //注册事件

}
