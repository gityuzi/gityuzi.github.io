sass学习一（基础）

发表于 2017-06-08 | 分类于 知识点 ， css

一直想学习的sass和less，了解过后选择了sass。此篇就是简单的sass入门笔记。

1. 简介

1.1 css预处理器

css预处理器包括：sass、less、stylus等。

为CSS增加可编程性，使得更加简洁，适应性，可读性，易于维护等优点

1.2 sass和scss

文件后缀名不一样：.sass/.scss

写法不一样：sass更为严格，不存在{}和；，scss和css的编写没有差别，可以将.css文件直接改为.scss。

2. 语法格式及编译

2.1 语法格式

将同一段css代码编写

body {
    font: 100% Helvetica, sans-serif;
    color: #333;
}

2.1.1 sass

$font-stack: Helvetica, sans-serif
$primary-color: #333

body
    font: 100% $font-stack
    color: $primary-color

2.1.2 scss

$font-stack: Helvetica, sans-serif;
$primary-color: #333;

body {
    font: 100% $font-stack;
    color: $primary-color;
}

sass由tab严格控制缩进，且没有{}和；的出现 ，而scss更像css代码。

2.2 编译

2.2.1 编译方法

  • 命令编译
  • GUI工具编译
  • 自动化编译

2.2.2 各个方法比较

编译名称 编译方法 编译优缺点 备注
命令编译 单个文件：sass <要编译的Sass文件路径>/style.scss:<要输出CSS文件路径>/style.css 多文件：sass sass/:css/ 只能一次性编译， 开启“watch”功能，这样只要你的代码进行任保修改，都能自动监测到代码的变化，并且给你直接编译出来 exp： sass –watch <要编译的Sass文件路径>/style.scss:<要输出CSS文件路径>/style.css
GUI界面工具编译 1.Koala (http://koala-app.com/)2.Compass.app（http://compass.kkbox.com/）3/Scout（http://mhs.github.io/scout-app/）4.**CodeKit**（https://incident57.com/codekit/index.html）5.Prepros（https://prepros.io/） 界面操作，比较方便。 无
自动化编译 grunt： glup 代码见下

grunt：

module.exports = function(grunt) {
    grunt.initConfig({
        pkg: grunt.file.readJSON('package.json'),
        sass: {
            dist: {
                files: {
                    'style/style.css' : 'sass/style.scss'
                }
            }
        },
        watch: {
            css: {
                files: '**/*.scss',
                tasks: ['sass']
            }
        }
    });
    grunt.loadNpmTasks('grunt-contrib-sass');
    grunt.loadNpmTasks('grunt-contrib-watch');
    grunt.registerTask('default',['watch']);
}

glup ：

var gulp = require('gulp');
var sass = require('gulp-sass');

gulp.task('sass', function () {
    gulp.src('./scss/*.scss')
        .pipe(sass())
        .pipe(gulp.dest('./css'));
});

gulp.task('watch', function() {
    gulp.watch('scss/*.scss', ['sass']);
});

gulp.task('default', ['sass','watch']);

2.2.3 常见编译错误

  • 字符：不支持GBK。需要设置为utf-8
  • 中文字符：建议文件命名用英文

2.2.4 输出方法

  • 嵌套输出方式 nested
  • 展开 expanded
  • 紧凑 compact

  • 压缩 compressed

    nav {

    ul {
    margin: 0;
    padding: 0;
    list-style: none;
}

li { display: inline-block; }

    }

nested ：

sass --watch test.scss:test.css --style nested

编译结果:

nav ul {
    margin: 0;
    padding: 0;
    list-style: none; }
nav li {
    display: inline-block; }
nav a {
    display: block;
    padding: 6px 12px;
    text-decoration: none; }

expanded ：

sass --watch test.scss:test.css --style expanded

结果：

nav ul {
    margin: 0;
    padding: 0;
    list-style: none;
}
nav li {
    display: inline-block;
}
nav a {
    display: block;
    padding: 6px 12px;
    text-decoration: none;
}

compacted：

sass --watch test.scss:test.css --style compact

结果：

nav ul { margin: 0; padding: 0; list-style: none; }
nav li { display: inline-block; }
nav a { display: block; padding: 6px 12px; text-decoration: none; }

compressed：

sass --watch test.scss:test.css --style compressed

结果：

nav ul{margin:0;padding:0;list-style:none}nav li{display:inline-block}nav a{display:block;padding:6px 12px;text-decoration:none}
